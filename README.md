# 资源库网站

这是一个美观的资源分享网站，用于管理和展示各种资源。

## 项目结构

```
├── index.html          # 主页面
├── css/               # 样式文件目录
│   └── style.css      # 主样式文件
├── js/                # JavaScript 文件目录
│   └── main.js        # 主脚本文件
└── data/              # 数据文件目录
    └── resources.json # 资源数据文件
```

## 功能特点

- 响应式设计，适配各种设备
- 资源分类展示
- 搜索功能
- 美观的卡片式布局
- 资源数据通过 JSON 文件管理

## 使用方法

1. 克隆项目到本地
2. 使用现代浏览器打开 index.html
3. 在 data/resources.json 中添加或修改资源数据

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- JSON
