# 资源共享平台

基于Node.js的轻量级资源分享网站，使用文件存储代替数据库

## 🚀 功能特性
- 资源卡片式展示
- 文件存储数据持久化
- 自动初始化示例数据
- 响应式网页设计

## 📦 安装步骤
1. 克隆仓库：
```bash
git clone https://github.com/qichenya/resource-library.git
```

2. 安装依赖：
```bash
cd resource-library && npm install
```

3. 创建数据存储目录：
```bash
mkdir data
```

## ⚙️ 配置说明
复制配置文件模板：
```bash
cp config.example.json config.json
```
